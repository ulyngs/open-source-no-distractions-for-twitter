This Safari extension hides distractions on Twitter. It is open source and collects zero data - you can find the source code on GitHub.

To turn it on, open Safari and go to twitter.com. Then:
1) In the toolbar tap 'Aa' -> 'Manage Extensions', and toggle the extension on.
2) Tap the extension and allow it to access twitter.com
